# HOTEL MAINTENANCE PRO - Architecture

## BUSINESS MODEL
- You: flooring-cleaning service provider (carpet, tile, both)
- Contracts: monthly sessions at a fixed price negotiated per hotel
- Bonus: a maintenance management app offered to hotel clients

## MAIN MODULES
1. CONTRACTS - Manage client contracts (fixed price, frequency)
2. DASHBOARD_PRO - Business view (revenue, sessions, issues)
3. DASHBOARD_HOTEL - Simplified hotel view (free)
4. MAINTENANCE - Report and manage issues

## PRICING STRUCTURE
- Reference: Carpet $45, Tile $40, Both $65 / room
- Contracts: negotiated fixed price (e.g. Grand Paris - 15 carpet rooms = $630/month)
- Included: floor cleaning + maintenance management via the app

## USER FLOW
1. Hotel reports an issue → Free app
2. You see the issue → Provider dashboard
3. You schedule a session → Contracts module
4. You execute → Worksheet generated
5. Hotel approves → Fully traceable
